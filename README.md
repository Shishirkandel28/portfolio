## 🚀 Getting Started

### Local Development

1. **Clone the repository:**
   ```bash
   git clone https://github.com/yourusername/portfolio-website.git
   cd portfolio-website
   ```

2. **Open in browser:**
   Simply open `index.html` in your web browser or use a local server:
   ```bash
   # Using Python
   python -m http.server 8000
   
   # Using Node.js
   npx serve .
   
   # Using PHP
   php -S localhost:8000
   ```

3. **View the website:**
   Open `http://localhost:8000` in your browser

### GitHub Pages Deployment

1. **Push to GitHub:**
   ```bash
   git add .
   git commit -m "Initial portfolio website"
   git push origin main
   ```

2. **Enable GitHub Pages:**
   - Go to repository Settings
   - Scroll to Pages section
   - Select "Deploy from a branch"
   - Choose "main" branch
   - Click Save

3. **Your site will be live at:**
   `https://yourusername.github.io/portfolio-website`

## 📝 Customization

### Personal Information
Update the following files with your information:
- Personal details in all HTML files
- Social media links in footer sections
- Profile images in the `images/` folder
- Contact information in `contact.html`

### Colors and Styling
Modify `css/style.css` to change:
- Color scheme (search for hex colors like `#2563eb`)
- Typography and fonts
- Layout and spacing
- Animation effects

### Adding New Projects
In `projects.html`:
1. Copy an existing project card
2. Update the image, title, description, and technologies
3. Add your project links

### Blog Posts
In `blog.html`:
1. Add new blog post cards
2. Update dates, titles, and content
3. Add appropriate tags and categories
